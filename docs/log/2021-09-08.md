## 系统服务加入开机启动

```
systemctl enable service-name
```

所有系统服务在目录`/lib/systemd/system`

## python 单下划线和双下划线的区别

- 单下划线，修饰模块成员和类成员，不可被`from M import *`导入，其它方面和公有一样
- 双下划线，修饰类成员，更高级别的私有，不可直接访问

## echarts 工具箱

```
toolbox: {
feature: {
    dataZoom: {},
    saveAsImage: {},
    dataView: {},
},
},
```

## echarts 改变 tooltip 位置防止遮挡

```
tooltip: {
    trigger: 'axis',
    position: function (pt) {
        return [pt[0]-40, '10%'];
    }
},
```
