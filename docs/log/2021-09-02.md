## python 生成登录 session

开始写后端的登录流程了，验证登录信息后，需要生成一个 session 加到 cookie 里

```python
import secrets

def create_session():
    session = secrets.token_urlsafe(16)
    return session # '-Mm80CqeR5GMf4zj9RFfGg'
```

## ADP 项目本地调试

dev 模式，请求本地测试服务器,配置 proxy.ts 即可

```json
  dev: {
    '/api/': {
      target: 'http://localhost:9000',
      changeOrigin: true,
      pathRewrite: { '^': '' },
    },
  },
```
